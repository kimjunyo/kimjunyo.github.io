#### **4. 코드 관련 문제 (Coding Questions)**

26. **아래 코드를 실행하면 출력되는 결과는 무엇인가?**

   ```java
   class Animal {
       void sound() {
           System.out.println("Animal makes a sound");
       }
   }

   class Dog extends Animal {
       @Override
       void sound() {
           System.out.println("Dog barks");
       }
   }

   public class Main {
       public static void main(String[] args) {
           Animal a = new Dog();
           a.sound();
       }
   }
   ```
